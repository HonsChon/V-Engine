# 资源文件夹

这个文件夹用于存放项目的资源文件，如纹理贴图和3D模型。

## 建议的文件结构

```
assets/
├── textures/
│   ├── albedo/
│   ├── normal/
│   ├── metallic/
│   ├── roughness/
│   └── ao/
└── models/
```

## 支持的格式

### 纹理
- PNG
- JPG/JPEG  
- TGA
- BMP

### 模型
- OBJ (计划支持)
- FBX (计划支持)
- glTF (计划支持)
